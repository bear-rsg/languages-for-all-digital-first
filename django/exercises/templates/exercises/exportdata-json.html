Export Data (JSON)

To download data from the LFA Digital First platform in JSON format, please use one of the following API endpoints:

  • /exercises/exportdata/json/studentscores/ - Student exercise attempts
  • /exercises/exportdata/json/exercises/ - Exercises
  • /exercises/exportdata/json/yeargroups/ - Year Groups
  • /exercises/exportdata/json/difficulties/ - Exercise Difficulties
  • /exercises/exportdata/json/languages/ - Languages
  • /exercises/exportdata/json/fontsizes/ - Font Sizes
  • /exercises/exportdata/json/schoolclasses/ - School Classes
  • /exercises/exportdata/json/themes/ - Exercise Themes
  • /exercises/exportdata/json/exerciseformats/ - Exercise Formats

All of these endpoints require the following parameters:

  • username - your username (typically your email address)
  • api_key - your API key (contact us if you don't know yours)

So a typical request may look like: /exercises/exportdata/json/studentscores/?username=example@email.com&api_key=myapikeygoeshere

If you have any issues please contact us for support.
